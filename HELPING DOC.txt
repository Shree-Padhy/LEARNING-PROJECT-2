How to run the Cloud Mart E-Commerce Website using PHP and MySQL
1. Download the  zip file

2. Extract the file and copy Cloud Mart folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name Cloud Mart3

6. Import Cloud Mart file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/Cloud Mart FINAL/User Panel (For User)

7.Run the script http://localhost/Cloud Mart FINAL/User Panel/admin (For Admin)
